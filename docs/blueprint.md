# **App Name**: MasseurPro

## Core Features:

- AI Demand Forecasting: Use machine learning to predict massage demand in various cities based on population, LGBTQ+ index, and real-time trends.  This demand tool is a key factor for therapists planning travel or relocation.
- Competitor Analysis: Analyze the competition by assessing competitor count, market saturation, and average service pricing in different areas.
- Personalized Trip Planning: Optimize rates based on AI pricing guidance considering demand, seasonality, and competition, aiming to maximize revenue from trips.
- Client Relationship Management (CRM): Implement basic CRM functions that capture repeat rate and customer reviews to improve user services.
- User Authentication: Basic email/password login with admin/demo accounts and the ability to toggle dark mode.
- Interactive Market Heatmap: Display a geographical representation of demand spikes, helping professionals spot key locations. This visualization tool highlights promising areas, displayed on an interactive and attractive heatmap.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey trust, wisdom, and luxury, suitable for professional service applications.
- Background color: Light gray (#F0F0F3), providing a clean, unobtrusive backdrop that emphasizes content. Suitable for a light scheme.
- Accent color: Soft lavender (#D8C2FF), to add a touch of elegance and interest.
- Body font: 'PT Sans', a versatile humanist sans-serif suitable for clear, readable body text. Headline font: 'PT Sans', using increased weight/size to create distinction. The combination ensures legibility and a clean aesthetic.
- Use consistent, professional icons related to massage therapy and market analysis to quickly convey meaning and aid navigation.
- Use a clean, responsive grid layout to effectively present market data and analysis in a structured and accessible manner.
- Apply subtle, tasteful animations to transitions and data updates for smooth UX.
